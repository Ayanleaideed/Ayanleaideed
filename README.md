## ğŸ¯ Getting Started

### Prerequisites

- Node.js (v14.0+)
- npm (v6.0+)
- Docker (optional, for containerized deployment)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/project-nebula/nebula.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Configure your environment variables
   ```sh
   cp .env.example .env
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
